# Regression Evaluation

In the below, the y(i) is actual value and y(hat i) is the predicted
value.

## Mean Absolute Error

```
MAE = 1/n ∑ | y(i) - y(hat i)|
```

## Mean Squared Error

```
MSE = 1/n ∑ ( y(i) - y(hat i)) ** 2
```

## Root Mean Squared Error

```
RMSE = Sqrt (1/n ∑ ( y(i) - y(hat i)) ** 2)
```
